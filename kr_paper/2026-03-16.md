## 선정 논문

### 1) ‘과학기술학’ 관점에서의 AI Agent 사례 연구: ‘젠스파크’ 사례를 중심으로 (KCI)
- 링크: https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART003277864
- DOI: 페이지 내 미확인(불확실)
- 요약(약 500자): 본 연구는 AI 에이전트를 독립 기술이 아니라 인간·사회와 공동 구성되는 행위자로 해석하기 위해 라투르의 과학기술학(STS) 프레임을 적용한다. 2025년 이후 확산되는 에이전트 서비스 수용 양상을 젠스파크 사례로 분석해, 이용자가 판단·탐색·생성 권한을 에이전트에 위임하는 과정과 그 사회적 함의를 논의한다. 특히 성능 담론 뒤에 가려진 인프라 비용·자원 편중·블랙박스화 문제를 드러내며, 기술 낙관/비관의 이분법을 넘어 관계적 분석틀이 필요함을 제시한다.
- 핵심 기여: STS 기반 AI 에이전트 사회기술 분석 프레임 제안
- 키워드: #AIAgent #STS #Sociotechnical #Delegation #AISociety

### 2) 거대언어모델을 활용한 생성형 행위자 기반 모형 (KCI 우수등재)
- 링크: https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART003146173
- DOI: 페이지 내 미확인(불확실)
- 요약: 전통 ABM의 연역적 가정 한계를 보완하기 위해 LLM 기반 생성형 행위자 모형을 제시하고, 단순/집단/개인 LLM 행위자 수준을 비교한다. 쉘링 분리모형 맥락에서 규칙기반 ABM과 LLM-ABM의 차이를 검토해, 현실성 있는 미시행동 생성 가능성과 동시에 재현성·통제성 저하 위험을 함께 지적한다. 사회과학 모형에 LLM을 도입할 때 설명가능성, 실험 반복성, 파라미터 고정 프로토콜이 필수라는 점을 보여준다.
- 핵심 기여: LLM-ABM의 가능성과 한계를 균형 있게 체계화
- 키워드: #LLM #AgentBasedModel #GenerativeAgents #SocialSimulation #Evaluation

### 3) 생성형 인공지능의 프롬프트 글쓰기 분석 (DBpia)
- 링크: https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE11758668
- DOI: 페이지 내 미확인(불확실)
- 요약: 프롬프트를 단순 입력문이 아니라 인간-기계 간 명령/의미를 매개하는 언어학적 기호 체계로 해석하고, 프롬프트의 배열 규칙과 문법 형성을 분석한다. 논문은 프롬프트 커뮤니티에서 읽기·쓰기·공유 관행이 일종의 랑그로 작동하며, 사용자 개별 실천은 파롤로 나타난다는 해석을 제시한다. 이는 prompt engineering을 기술 최적화 문제만이 아니라 사회적 글쓰기 실천으로 확장해 이해할 필요가 있음을 시사한다.
- 핵심 기여: 프롬프트 엔지니어링의 언어기호학적 분석 프레임 제공
- 키워드: #PromptEngineering #GenerativeAI #Writing #HumanAIInteraction #Semiotics

### 4) 생성형 AI 모델을 활용한 비즈니스 데이터 분석 리포트 생성 프롬프트 엔지니어링 연구 (DBpia)
- 링크: https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE11825507
- DOI: 페이지 내 미확인(불확실)
- 요약: 퍼포먼스 마케팅 사례를 대상으로 LLM 기반 보고서 생성 프롬프트 구조를 설계하고, ROUGE·BERTScore와 체크리스트 평가를 병행해 성능을 검증한다. 결과적으로 구조화 프롬프트가 정확도와 업무 속도를 개선할 잠재력을 보였으며, 업무형 문서 생성에서 프롬프트 템플릿 표준화 필요성을 제기한다. 다만 학술대회 성격상 표본·도메인 범위가 제한될 수 있어 타 산업군 재현 검증이 필요하다(불확실성).
- 핵심 기여: 데이터 분석 보고서 자동화용 평가 가능한 프롬프트 설계
- 키워드: #DataScience #PromptEngineering #BusinessAnalytics #LLM #ReportGeneration

### 5) 매치3 게임 플레이를 위한 PPO 알고리즘 강화학습 에이전트 설계/구현 (RISS, KCI)
- 링크: https://m.riss.kr/search/detail/DetailView.do?p_mat_type=1a0202e37d52c72d&control_no=3d367d0d757ac9ff47de9c1710b0298d
- DOI: http://dx.doi.org/10.22156/CS4SMB.2021.11.03.001
- 요약: 유니티 ML 환경에서 PPO 기반 강화학습 에이전트를 구현해 매치3 퍼즐 자동 플레이 성능을 검증한 연구다. 대화형 LLM 에이전트와는 과제가 다르지만, 상태-행동-보상 설계·탐색전략·학습 안정화라는 핵심 에이전트 공학 요소를 명확히 다룬다. 최근 LLM 에이전트의 tool reliability 문제를 다룰 때도 RL 관점의 정책 안정성, 실패 회복, 환경 시뮬레이션 평가가 연결될 수 있다는 점에서 방법론적 참고 가치가 있다.
- 핵심 기여: 실험 가능한 에이전트 학습 파이프라인 제시
- 키워드: #ReinforcementLearning #AIAgent #PPO #GameAI #Reliability
