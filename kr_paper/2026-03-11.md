### 1) 과학 학습의 어려움 진단을 위한 LLM 기반 대화형 에이전트 설계 및 평가 (KCI: ART003228306)
- 링크: https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART003228306
- Google Scholar 추적(보조): https://scholar.google.com/scholar?q=%EA%B3%BC%ED%95%99+%ED%95%99%EC%8A%B5%EC%9D%98+%EC%96%B4%EB%A0%A4%EC%9B%80+%EC%A7%84%EB%8B%A8%EC%9D%84+%EC%9C%84%ED%95%9C+LLM+%EA%B8%B0%EB%B0%98+%EB%8C%80%ED%99%94%ED%98%95+%EC%97%90%EC%9D%B4%EC%A0%84%ED%8A%B8
- DOI: 확인 불가(플랫폼 페이지에서 명시 미확인)
- 요약(약 500자): 본 연구는 과학 학습 곤란 진단을 위해 LLM 대화형 에이전트 4개 프로토타입(분류 방식·정서적 지지 유무 조합)을 설계하고, ChatGPT-4o 기반 시뮬레이션 및 실제 중3 학생 적용 실험을 수행했다. 결과적으로 환경요인 세분 모델이 개인요인 세분 모델보다 진단 정확도가 높았고, 정서적 지지 추가는 통계적으로 유의한 차이를 보이지 않았다. 실제 적용에서는 지역사회 요인 누락, 학생의 주제 이탈/회피 시 조기 종료 문제가 관찰되어, 진단 대화 설계에서 탐색 지속성·복귀 유도 프롬프트·누락 점검 루프가 핵심 설계요소임을 제시한다.
- 핵심 기여: 진단형 에이전트 설계 비교실험, 교육현장 적용 시 실패패턴 식별, 개선 가이드 제시
- #키워드: #KCI #LLM #ConversationalAgent #PromptEngineering #EducationAI

### 2) 거대언어모델을 활용한 생성형 행위자 기반 모형 (KCI: ART003146173, 한국사회학 58(4), 2024)
- 링크: https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART003146173
- DOI: 확인 불가(페이지 노출 범위 내 미확인)
- 요약: 전통 ABM의 연역적 규칙 가정 한계를 보완하기 위해, LLM 기반 생성형 행위자(단순/집단/개인) 접근을 제시하고 쉘링 분리모형 맥락에서 비교 논의한다. 연구의 핵심은 미시 규칙을 고정하는 대신, LLM의 인간 유사 추론을 통해 보다 귀납적·현실적 시뮬레이션을 구축할 수 있는 가능성을 검토한 점이다. 동시에 재현성·통제성·편향 전이 같은 방법론적 위험도 병행 제시해, 사회과학적 AI 에이전트 활용에서 ‘현실성 증가’와 ‘실험 엄밀성 유지’ 간 균형 문제를 선명히 드러낸다.
- 핵심 기여: LLM+ABM 통합 프레임, 행위자 복잡도 단계화, 사회과학 시뮬레이션 확장 가능성 제시
- #키워드: #KCI #GenerativeAgent #ABM #SocialSimulation #LLM

### 3) 대화형 텍스트 기반 게임 자동화를 위한 LLM 기반 에이전트 비교 연구 (KCI: ART003214571)
- 링크: https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART003214571
- DOI: 확인 불가
- 요약: Zork I 자동 플레이 과제를 통해 GPT-4·Gemini·DeepSeek 기반 에이전트를 비교했고, Stateless 한계를 보완하기 위해 상태저장(State Manager) 모듈을 프롬프트 생성 루프에 결합했다. 평가지표는 점수, 고유 장소 방문 수, 반복 명령 수 등 정량 지표와 행동 특성 정성 분석으로 구성되었다. 결과는 모델별 강점이 분화됨을 보여주며(탐험성 vs 점수 최적화), 상태정보 관리가 무의미 명령 감소와 전략 일관성 향상에 유효함을 시사한다. 이는 에이전트 메모리 설계가 단순 저장이 아니라 ‘행동 정책 안정화’ 장치임을 보여준다.
- 핵심 기여: 게임 기반 에이전트 벤치마크, 상태관리 모듈 효과 검증, 모델별 전략 특성 분석
- #키워드: #KCI #LLMAgent #StateManager #Evaluation #GameAI

### 4) Beyond Text: Implementing Multimodal LLM-Powered Multi-Agent Systems Using a No-Code Platform (DBpia: NODE12127075)
- 링크: https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE12127075
- Google Scholar 추적(보조): https://scholar.google.com/scholar?q=Beyond+Text+Implementing+Multimodal+Large+Language+Model-Powered+Multi-Agent+Systems+Using+a+No-Code+Platform
- DOI: 확인 불가(메타데이터 비노출)
- 요약: 기업 내 AI 도입 장벽(기술복잡도·구현비용·개발인력 의존)을 낮추기 위해 No-Code 기반 멀티모달 멀티에이전트 시스템을 제안한다. 이미지 메모 기반 코드 생성, Advanced RAG 질의응답, 텍스트-이미지-영상 연계 생성 등 실무 시나리오를 통해 비개발자도 AI 파이프라인을 운영할 수 있음을 보였다. 연구는 멀티에이전트 도입을 모델 성능 경쟁이 아닌 운영 접근성·조직 확산의 문제로 전환해 제시한다는 점에서 실무 가치가 크다. 다만 공개된 페이지 정보만으로는 정량 성능지표 세부값 확인이 제한된다.
- 핵심 기여: No-Code 기반 멀티에이전트 실무 프레임, 멀티모달 워크플로우 제시, 조직 도입성 강화
- #키워드: #DBpia #MultimodalAgent #NoCode #EnterpriseAI #RAG

- DOI: 확인 불가
- 요약: AI 스피커 사용자 416명을 대상으로, VUI 에이전트의 음성품질(친밀감·정서표현·이해가능성·반응성 등)이 사회적 실재감, 지각가치, 지속사용의도에 미치는 구조를 실증 분석했다. 주요 결과는 친밀감/정서표현/이해가능성/반응성이 사회적 실재감에 유의한 정(+) 영향을 보였고, 사회적 실재감은 지각가치와 지속사용의도를 강화했다는 점이다. 일부 요인(예: 명료성·자연스러움)은 기대와 다른 효과를 보여, 에이전트 설계에서 음성 기술 품질과 관계형 상호작용 품질을 분리 설계해야 함을 시사한다.
