## 1. Interpretable Context Methodology: Folder Structure as Agentic Architecture (MWP)

**요약:** AI 에이전트 오케스트레이션에서 프레임워크 수준의 코드 기반 조정 대신 파일 시스템 구조를 활용하는 Model Workspace Protocol(MWP)을 제안한다. 번호가 매겨진 폴더가 단계를 나타내고, 일반 마크다운 파일이 각 단계에서 단일 AI 에이전트에게 전달할 프롬프트와 컨텍스트를 담는다. Unix 파이프라인 설계, 모듈 분해, 리터레이트 프로그래밍 원칙을 AI 에이전트 컨텍스트 구조화에 적용한다. MIT 라이선스 오픈소스.

**핵심기여:** 프레임워크 없이 파일시스템 구조만으로 순차적 워크플로를 구성하는 새로운 컨텍스트 엔지니어링 패러다임 제시

## 2. Efficient LLM Safety Evaluation through Multi-Agent Debate (HAJailBench)

**요약:** LLM 안전 평가에서 비용 문제를 해결하기 위해 구조화된 멀티 에이전트 디베이트 프레임워크를 제안한다. 비평가(critic), 방어자(defender), 판사(judge) 에이전트가 공통 안전 루브릭 하에 디베이트를 수행한다. 11,100개의 인간 주석이 달린 jailbreak 벤치마크(HAJailBench)를 구축하여 평가를 진행했으며, 소형 모델 베이스라인 및 이전 멀티 에이전트 판사를 상회하는 성능을 달성하면서도 GPT-4o보다 경제적인 운영 비용을 실현했다.

**핵심기여:** 인간 주석 벤치마크 + 멀티 에이전트 디베이트 기반 확장 가능한 LLM 안전 평가 파이프라인

## 3. OMNIFLOW: A Physics-Grounded Multimodal Agent for Generalized Scientific Reasoning

**요약:** LLM이 편미분방정식(PDE) 기반의 연속적 시공간 역학을 다룰 때 비물리적 환각(hallucination)을 일으키는 문제를 해결하기 위해 신경-기호(neuro-symbolic) 아키텍처 OMNIFLOW를 제안한다. Semantic-Symbolic Alignment 메커니즘이 고차원 유동 텐서를 위상적 언어 기술자(topological linguistic descriptors)로 투영하여 모델이 원시 픽셀이 아닌 물리적 구조를 인지하도록 한다. Physics-Guided Chain-of-Thought(PG-CoT) 워크플로우가 질량 보존 등 물리적 제약 조건을 주입한 반복적 추론을 수행한다.

**핵심기여:** 도메인 특화 파인튜닝 없이 프리즌(frozen) 멀티모달 LLM을 물리 법칙에 정합(ground)시키는 신경-기호 프레임워크

**키워드:** #MultimodalAgent #NeuroSymbolic #PhysicsReasoning #ScientificAI #GroundedReasoning

## 5. An Agentic Multi-Agent Architecture for Cybersecurity Risk Management

**요약:** 소규모 조직의 사이버보안 위험 평가 비용($15,000 이상, 수주 소요) 문제를 해결하기 위해 6개 에이전트 AI 시스템을 구축했다. 각 에이전트는 조직 프로파일링, 자산 매핑, 위협 분석, 통제 평가, 위험 점수화, 권고사항 생성을 담당하며, 지속적 컨텍스트를 공유하여 후속 에이전트가 선행 결론을 기반으로 작업한다. ## 6. Memori: A Persistent Memory Layer for Efficient, Context-Aware LLM Agents

**요약:** LLM 에이전트를 위한 LLM-agnostic 영구 메모리 레이어 Memori를 제안한다. 기존 접근법이 대량의 원시 대화를 프롬프트에 주입하는 반면, Memori는 메모리를 데이터 구조화 문제로 취급하여 비정형 대화를 컴팩트한 시맨틱 트리플과 대화 요약으로 변환한다. LoCoMo 벤치마크에서 81.95% 정확도를 달성하면서, 쿼리당 단 1,294 토큰(전체 컨텍스트의 ~5%)만 사용하여 경쟁 방식 대비 67% 토큰 절감, 전체 컨텍스트 방식 대비 20배 이상의 비용 절감을 실현했다.

**핵심기여:** 구조화된 메모리 표현이 더 큰 컨텍스트 윈도우보다 효과적임을 실증적으로 입증

## 9. MemMA: Coordinating the Memory Cycle through Multi-Agent Reasoning and In-Situ Self-Evolution

**요약:** 메모리 증강 LLM 에이전트에서 메모리 구축, 검색, 활용을 분리된 서브루틴으로 처리하는 기존 시스템의 한계를 지적한다. MemMA는 플러그앤플레이 멀티 에이전트 프레임워크로 순방향 경로(forward path)에서 Meta-Thinker가 구조화된 지침을 생성하여 Memory Manager와 Query Reasoner를 조정하고, 역방향 경로(backward path)에서 probe QA 합성 및 검증을 통한 자기 진화적 메모리 구축을 수행한다. LoCoMo에서 다양한 LLM 백본과 스토리지 백엔드에 대해 일관되게 기존 베이스라인을 상회한다.

**핵심기여:** 메모리 사이클의 순/역방향 경로를 통합적으로 조정하는 자기 진화적 메모리 관리 프레임워크

**키워드:** #MemoryCycle #MultiAgentReasoning #SelfEvolution #LoCoMo #PlugAndPlay

## 10. Governed Memory: A Production Architecture for Multi-Agent Workflows

**요약:** 엔터프라이즈 AI에서 수십 개의 자율 에이전트가 동일 엔티티에 대해 공유 메모리 없이 독립적으로 작동하여 발생하는 메모리 거버넌스 갭(실로화, 파편화, 비구조화, 중복 전달, 무음 품질 저하)을 식별하고 Governed Memory를 제안한다. 개방형 원자 사실과 스키마 강제 타입 속성을 결합하는 듀얼 메모리 모델, 계층적 거버넌스 라우팅, 반사 경계 검색, 폐루프 스키마 라이프사이클을 포함한다. Personize.ai에서 프로덕션 운영 중이며, LoCoMo에서 74.8% 정확도 달성.

## 11. VISTA: Reflection in the Dark — Exposing and Escaping the Black Box in Reflective Prompt Optimization

**요약:** GEPA 등 반사적 자동 프롬프트 최적화(APO) 방법이 최적화 과정을 블랙박스로 유지하여 해석 불가능한 궤적과 체계적 실패를 초래하는 문제를 다룬다. VISTA는 가설 생성과 프롬프트 재작성을 분리(decouple)하여 시맨틱 라벨링된 가설, 병렬 미니배치 검증, 해석 가능한 최적화 추적을 가능하게 한다. 2계층 탐색-활용 메커니즘이 국소 최적해 탈출에 기여한다. GSM8K에서 결함 시드에 대해 GEPA가 23.81% → 13.50%로 성능 저하를 보인 반면, VISTA는 87.57%로 회복했다.

**핵심기여:** APO 프로세스의 블랙박스 문제를 해결하고 국소 최적해 탈출을 위한 멀티 에이전트 프레임워크

**키워드:** #PromptOptimization #APO #VISTA #BlackBox #MultiAgent

## 12. When Prompt Optimization Becomes Jailbreaking: Adaptive Red-Teaming of LLMs (EACL SRW 2026 Oral)

**요약:** 고정된 해로운 프롬프트 모음에 의존하는 기존 안전 평가가 적응형 공격을 간과한다는 한계를 지적한다. DSPy를 사용하여 원래 양성(benign) 작업 성능 향상을 위해 설계된 블랙박스 프롬프트 최적화 기법을 안전 실패 탐색에 전용(repurpose)한다. GPT-5.1을 평가자로 사용하여 HarmfulQA 및 JailbreakBench 프롬프트를 최적화한 결과, Qwen 3 8B의 평균 위험 점수가 0.09 → 0.79로 급증하여 정적 벤치마크가 잔여 위험을 과소평가함을 입증했다.

**핵심기여:** 프롬프트 최적화 → jailbreak 전용의 전환 가능성 규명 및 적응형 레드팀핑의 필요성 제기
