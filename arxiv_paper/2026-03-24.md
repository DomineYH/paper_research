## 1. MemoryAgentBench: LLM Agent 메모리 평가를 위한 점진적 멀티턴 상호작용 벤치마크

**핵심 요약:**  
기존 LLM 에이전트 벤치마크는 추론·계획·실행 능력에 집중했으나, 메모리(기억·갱신·검색) 평가는 부족했다. 본 연구는 인지과학 이론을 기반으로 4가지 핵심 메모리 역량(정확한 검색, 테스트 타임 학습, 장거리 이해, 선택적 망각)을 정의하고, 이를 포괄하는 MemoryAgentBench를 제안한다. 기존 long-context 데이터셋을 멀티턴 형식으로 변환하여 에이전트의 점진적 정보 처리를 시뮬레이션한다. RAG 시스템부터 외부 메모리 모듈을 갖춘 고급 에이전트까지 다양한 메모리 에이전트를 평가한 결과, 현재 방법론은 4가지 역량을 모두 완벽히 달성하지 못하는 것으로 나타났다.

**핵심 기여:** 메모리 전용 벤치마크 최초 제안; 4가지 인지적 메모리 역량 정의; 기존 long-context QA 데이터셋을 멀티턴 메모리 평가로 재설계

**링크:** <https://arxiv.org/abs/2507.05257>  
**DOI:** 10.48550/arXiv.2507.05257  
**해시태그:** #LLMAgent #AgentMemory #Benchmark #RAG #MultiTurn #CognitiveScience

## 2. Helium: 에이전트 워크플로우를 위한 효율적 LLM 서빙 (데이터 시스템 관점)

**핵심 요약:**  
에이전트 워크플로우는 상호 의존적인 LLM 호출 시퀀스로 구성되며, 중복 프롬프트와 중간 결과로 인해 광범위한 중복이 발생한다. 기존 vLLM 등 LLM 서빙 시스템은 개별 추론 호출 최적화에 집중하고 호출 간 의존성을 간과한다. 본 연구는 에이전트 워크로드를 쿼리 플랜으로 모델링하고 LLM 호출을 일급 연산자로 취급하는 Helium 프레임워크를 제안한다. 선제적 캐싱과 캐시 인식 스케줄링을 통합하여 프롬프트, KV 상태, 워크플로우 전반의 재사용을 극대화하며, 최대 1.56배 속도 향상을 달성한다.

**핵심 기여:** 에이전트 워크플로우를 쿼리 최적화 관점으로 재해석; 크로스-콜 의존성 기반 캐싱 설계; 기존 SOTA 대비 최대 1.56배 속도 향상

**링크:** <https://arxiv.org/abs/2603.16104>  
**DOI:** 10.48550/arXiv.2603.16104  
**해시태그:** #LLMServing #AgenticWorkflow #QueryOptimization #Caching #DataSystems

## 3. UniSAFE: 통합 멀티모달 모델을 위한 종합 안전 평가 벤치마크

**핵심 요약:**  
통합 멀티모달 모델(UMM)은 강력한 교차 모달리티 능력을 제공하지만, 단일 작업 모델에서 관찰되지 않는 새로운 안전 위험을 도입한다. 본 연구는 7가지 I/O 모달리티 조합, 6,802개 인스턴스로 구성된 UniSAFE 벤치마크를 제안한다. 공유 타겟 설계로 공통 위험 시나리오를 다양한 I/O 구성에 투영하여 제어된 교차 작업 비교를 가능하게 한다. 15개 SOTA UMM 평가 결과, 다중 이미지 구성 및 멀티턴 설정에서 안전 위반이 크게 증가하며, 이미지 출력 작업이 텍스트 출력보다 취약함을 확인했다.

**핵심 기여:** UMM용 최초 시스템 수준 안전 벤치마크; 7가지 I/O 모달리티 조합 포괄; 15개 프롭리어터리/오픈소스 모델 평가

**링크:** <https://arxiv.org/abs/2603.17476>  
**DOI:** 확인 중  
**해시태그:** #MultimodalSafety #AIAlignment #Benchmark #UMM #SafetyEvaluation

## 4. SF-RAG: 학술 질의응답을 위한 구조 충실도 RAG

**핵심 요약:**  
기존 RAG 접근법은 논문을 비구조적 청크로 평탄화하여 계층 구조를 파괴하고, 단편화된 컨텍스트와 토큰 오배분을 초래한다. SF-RAG는 학술 논문의 기본 계층 구조를 저엔트로피 검색 사전으로 활용하는 RAG 프레임워크다. 구조 충실도 인덱스를 구성하고, 경로 안내 검색 기법으로 쿼리 의미를 관련 섹션에 정렬하여 고관련성 root-to-leaf 경로를 선택한다. 엔트로피 기반 구조 진단법으로 검색 단편화와 증거 할당 정확도를 정량화하며, 3개 QA 벤치마크에서 검색 단편화 감소 및 응답 품질 향상을 입증했다.

**핵심 기여:** 논문의 기본 계층 구조를 검색 사전으로 활용; 엔트로피 기반 구조 진단 지표 제안; 기존 RAG 대비 증거 할당 정확도 개선

**링크:** <https://arxiv.org/abs/2602.13647>  
**DOI:** 확인 중  
**해시태그:** #RAG #AcademicQA #StructuredRetrieval #Entropy #DocumentIntelligence

## 5. AgentDS: 도메인 특화 데이터 사이언스에서의 인간-AI 협업 벤치마크

**핵심 요약:**  
AI 에이전트가 도메인 특화 데이터 사이언스 작업에서 인간 전문가 수준에 도달할 수 있는지 불확실하다. AgentDS는 6개 산업(상거래, 식품 생산, 의료, 보험, 제조, 소매 은행)의 17개 챌린지로 구성된 벤치마크 및 대회다. 29팀 80명 참가자를 통한 개방 대회를 진행하여 AI 전용 솔루션과 인간-AI 협업을 체계적으로 비교했다. 핵심 발견: (1) 자율 에이전트는 도메인 특화 추론에 어려움 겪음, (2) 인간-AI 협업이 AI 전용 솔루션보다 우수, (3) 효과적 협업을 위한 인터페이스 설계가 핵심 과제.

**핵심 기여:** 산업 규모 멀티 도메인 데이터 사이언스 벤치마크; 29팀 대회 기반 실증적 인간-AI 협업 비교; 자율 에이전트 한계와 인간 전문가의 가치 입증

## 6. 다중 에이전트 아키텍처를 활용한 사이버보안 위험 관리

**핵심 요약:**  
소규모 조직의 사이버보안 위험 평가는 비용($15,000 이상), 시간(수주), 인력 부족으로 대부분 건너뛰어진다. 본 연구는 6개 에이전트(조직 프로파일링, 자산 매핑, 위협 분석, 통제 평가, 위험 평점, 권고 생성)로 구성된 AI 시스템을 구축했다. 에이전트 간 지속적 컨텍스트 공유로 이전 단계 결론을 기반 구축하며, HIPAA 적용 의료 기업에서 CISSP 실무자 3인과 85% 심각도 분류 일치, 92% 위험 식별 커버리지를 달성했다. 중요 발견: Tesla T4(4,096 토큰)에서 전체 멀티에이전트 파이프라인은 30회 모두 실패하여 컨텍스트 용량이 모델 품질보다 결정적 병목임을 증명했다.

**핵심 기여:** 6-에이전트 구조화 보안 평가 파이프라인; CISSP 전문가와 85% 일치도 달성; 컨텍스트 윈도우가 에이전트 배포의 핵심 병목임을 실증

**링크:** <https://arxiv.org/abs/2603.20131>  
**DOI:** 확인 중

## 7. Caging the Agents: 의료 분야 자율 AI를 위한 제로 트러스트 보안 아키텍처

**핵심 요약:**  
의료 환경에서 자율 AI 에이전트는 HIPAA 위반 가능성을 내포한다. 본 연구는 의료 기술 기업의 9개 프로덕션 AI 에이전트를 위해 6-도메인 위협 모델을 개발하고 4계층 방어 전략을 구현했다: (1) gVisor 커널 수준 격리, (2) 크리덴셜 프록시 사이드카, (3) 네트워크 송신 정책, (4) 프롬프트 무결성 프레임워크. 90일 배포 결과, 자동 보안 감사 에이전트가 4개 HIGH 심각도 취약점을 발견·조치했으며, 문헌의 11가지 공격 패턴 전체에 대한 방어 커버리지를 달성했다. 모든 설정과 도구는 오픈소스로 공개된다.

**핵심 요약:**  
모놀리식 LLM에서 모듈형 스킬 장착 에이전트로의 전환은 AI 배포의 근본적 아키텍처 변화다. 본 연구는 GitHub 오픈소스 저장소 마이닝을 통한 자동화된 에이전트 스킬 획득 프레임워크를 제안한다. TheoremExplainAgent와 Code2Video(Manim 수학 애니메이션 엔진 기반)에서 시각화 및 교육 능력을 추출하고, SKILL.md 표준 포맷으로 변환하는 파이프라인을 구현했다. 에이전트 생성 교육 콘텐츠는 지식 전달 효율 40% 향상을 달성하면서도 인간 제작 튜토리얼과 동등한 교육학적 품질을 유지했다.

## 9. LLM 정책 합성에서의 협력과 착취: 순차적 사회적 딜레마

**핵심 요약:**  
본 연구는 LLM을 사용해 다중 에이전트 환경의 프로그래밍 에이전트 정책을 반복적으로 생성·평가·개선하는 LLM 정책 합성을 연구한다. 피드백 엔지니어링(스파스 vs 밀집 피드백)을 조사한 결과, Claude Sonnet 4.6과 Gemini 3.1 Pro 모두 밀집 피드백(효율성, 평등, 지속가능성, 평화 메트릭 포함)이 스파스 피드백(스칼라 보상만)을 지속적으로 초과하거나 동등한 성능을 보였다. 또한 LLM이 환경을 보상 해킹할 수 있는지 적대적 실험을 수행하여 5가지 공격 클래스를 특정하고, 표현력과 안전 간의 근본적 긴장을 논의한다.

**핵심 기여:** 피드백 엔지니어링이 에이전트 협력 전략에 미치는 영향 체계적 분석; 5가지 보상 해킹 공격 클래스 특정; LLM 정책 합성에서 표현력-안전 트레이드오프 논의

## 10. Anonymous-by-Construction: 프라이버시 보존 텍스트를 위한 LLM 구동 프레임워크

**핵심 요약:**  
AI의 책임 있는 사용을 위해 PII 보호가 필수적이다. 본 연구는 온프레미스 로컬 LLM 기반 치환 파이프라인으로, PII를 현실적이고 유형 일치하는 대체물로 대체하여 텍스트를 익명화한다. Microsoft Presidio, Google DLP, ZSTS 등 기존 방법과 비교하여 프라이버시, 의미론적 효용, 사실적 효용, 훈련 가능성(LoRA 미세조정)을 통합 평가했다. 에이전트 Q&A 성능도 평가하여 익명화 계층이 답변 품질에 미치는 영향을 측정했다. 결과적으로 로컬 LLM 치환이 규칙 기반 접근법과 NER 베이스라인을 초월하는 프라이버시-효용 프론티어를 달성했다.

**핵심 기여:** 온프레미스 LLM 기반 PII 치환 파이프라인; 프라이버시·효용·훈련 가능성 통합 평가 프로토콜; 에이전트 Q&A 익명화 계층 성능 평가

**링크:** <https://arxiv.org/abs/2603.17217>  
**DOI:** 확인 중  
**해시태그:** #Privacy #LLMAnonymization #AgenticQA #DataSecurity #PII
