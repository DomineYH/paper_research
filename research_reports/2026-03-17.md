# 심층 연구 보고서 (2026-03-17)

## 1) 박사학위급 연구 주제 도출
**주제 A (핵심):**
**"장기기억 기반 멀티에이전트 RAG 시스템의 신뢰성 평가 프레임워크: 시계열 업무환경과 고위험 도메인(국방/금융)에서의 통합 검증"**

- 도출 근거(당일 수집):
  - arXiv의 장기 메모리 압축/보존(Structured Distillation), 장기 개인화 평가(LifeSim), 커스텀 eval 생성(AgentFuel), 보안 고려(Security Considerations), 문서 탐색 전략 분석(Strategic Navigation).
  - 국내 문헌의 RAG 산업적용(국방 DELIIS), 멀티에이전트+RAG(금융 설명가능성), Graph Agent 기반 Advanced RAG, 국내 RAG 평가 동향.

## 2) 다각도 분석

### 2.1 이론(Theory)
- 문제정의: 현행 RAG/에이전트 연구는 (a) 컨텍스트 비용, (b) 장기기억 품질, (c) 평가 일관성, (d) 안전성 거버넌스를 분리해 다루는 경향이 있다.
- 제안 이론틀: **Memory-Eval-Safety Coupling (MESC)**
  1. Memory layer: 압축/검색 보존율
  2. Eval layer: 과업별(시계열·문서추론·개인화) 정량/정성 지표
  3. Safety layer: 권한·도구호출·출처검증·환각통제
- 가설(H): 메모리 비용최적화와 신뢰성은 독립 변수가 아니라 안전·평가 설계와 결합될 때만 실제 운영 이득을 낸다.

### 2.2 방법(Method)
- 시스템: 멀티에이전트 오케스트레이션(Planner/Retriever/Verifier/Reporter) + Graph Agent 기반 제어.
- 데이터:
  - 공개 문서 컬렉션 + 도메인 문서(국방/금융 유사 데이터셋; 실제 민감데이터는 비식별/샌드박스 전제).
  - 장기 대화 시나리오(선호변화·목표전환 포함) 생성.
- 실험 설계:
  - Baseline 1: 무메모리 RAG
  - Baseline 2: 원문대화 누적 메모리
  - Proposed: 구조화 증류 메모리 + 동적 평가 + 안전정책
- 측정:
  - 검색 보존율, 토큰비용, 응답 사실성, 출처 정합성, 장기 일관성, 안전위반율.

### 2.3 평가(Evaluation)
- 자동평가: task success, citation grounding score, latency/cost profile.
- 인간평가: 전문가 블라인드 평가(정확성·설명가능성·신뢰성).
- 반사실 실험: 메모리 압축율을 단계적으로 높여 성능 붕괴 지점 탐색.
- 불확실성 표기: 금일 수집 논문의 일부는 preprint/스니펫 기반이므로 세부 수치 비교에는 재현실험 필요.

### 2.4 윤리(Ethics)
- 개인정보/민감정보: 장기기억 설계 시 데이터 최소화·보존기간 제한 필요.
- 책임성: 에이전트 의사결정 로그 및 출처 체인 감사 가능해야 함.
- 안전성: 고위험 도메인에서 human-in-the-loop 승인 단계 필수.

### 2.5 산업(Industry)
- 국방/공공: 폐쇄망 RAG + 감사가능 로그 구조가 채택 포인트.
- 금융: 설명가능 멀티에이전트 설계와 규제 대응형 리포팅 필요.
- 엔터프라이즈 분석: 시계열 데이터 에이전트 eval 자동화 도입 가치 큼.

## 3) 실행 가능한 연구 질문(RQ)
- RQ1: 메모리 증류율 증가가 장기 개인화 성능에 미치는 임계점은 무엇인가?
- RQ2: Graph Agent 기반 검증 단계가 환각/오판 비용을 얼마나 줄이는가?
- RQ3: 산업 도메인별(국방/금융/일반문서) 평가 지표의 전이 가능성은 어느 정도인가?
